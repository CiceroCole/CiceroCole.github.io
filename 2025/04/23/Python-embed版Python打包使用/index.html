embed版Python就是精简纯净的Python，非常适合打包发布。

1.下载embed版Python

  • 通过华为源下载
  • 打开https://mirrors.huaweicloud.com/python/
  • 选取想要版本的Python，找到XXXXXX-embed-win32.zip后缀的文件下载
  • 将下载的文件解压就好

2.配置._pth文件与安装pip
